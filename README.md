## Tokens
The lexer recognizes the following tokens (names come from `src/tokenTypes.js`):

### Keywords
- CLASS
- EXTENDS
- INIT
- METHOD
- SUPER
- RETURN
- IF
- ELSE
- WHILE
- BREAK
- NEW
- THIS
- TRUE
- FALSE
- PRINTLN

### Types
- INT_TYPE
- BOOLEAN_TYPE
- VOID_TYPE

### Literals & Identifiers
- IDENTIFIER
- INTEGER
- STRING

### Punctuation & Operators
- LPAREN `(` 
- RPAREN `)`
- LBRACE `{`
- RBRACE `}`
- SEMICOLON `;`
- COMMA `,`
- DOT `.`
- ASSIGN `=`
- PLUS `+`
- MINUS `-`
- STAR `*`
- SLASH `/`

### Other
- EOF
